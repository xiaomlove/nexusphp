完整的 PT 建站解决方案。基于 NexusPHP + Laravel Framework + Element Plus。

## 功能特性
- 发种
- 特别区  
- 求种
- 字幕
- 考核
- H&R
- 签到
- 勋章
- 论坛 
- 多语言
- 自动备份
- 管理后台  
- Json api
- ....

## 系统要求
- PHP: 8.0，必须扩展：bcmath, ctype, fileinfo, json, mbstring, openssl, pdo_mysql, tokenizer, xml, mysqli, gd
- Mysql: 5.7最新版或以上版本
- Redis：可选
